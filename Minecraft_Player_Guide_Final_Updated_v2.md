# Minecraft 玩家手冊｜湖中暑8

## 1. 伺服器連線資訊

### Java 版玩家

請在「多人遊戲」新增伺服器，輸入：

```text
database-unpainted.gl.joinmc.link
```

### 基岩版 / 手機版玩家

請在「遊玩 > 伺服器 > 新增伺服器」中輸入：

```text
伺服器位址：traffic-convert.gl.at.ply.gg
連接埠：57476
```

支援平台：

- Minecraft Java Edition
- Windows 基岩版
- Android
- iPhone / iPad

---

## 2. 加入伺服器與白名單

伺服器平常會使用白名單保護。

如果尚未加入白名單，請先到 Discord 聯絡管理員，並提供：

```text
Minecraft 名稱：
版本：Java / 基岩版
```

管理員會在固定開放時段短暫關閉白名單，讓新玩家進入一次，確認名稱後再加入白名單。

注意：

- 沒有進入 Discord 或沒有聯絡管理員的玩家，可能無法加入伺服器。
- 基岩版玩家名稱進服後可能會顯示為 `.玩家名稱`。
- 白名單重新開啟後，只有已加入名單的玩家可以進入。

---

## 3. BlueMap 網頁地圖

伺服器支援 BlueMap 網頁地圖，可用瀏覽器查看：

- 玩家即時位置
- 主世界地圖
- 地獄地圖
- 終界地圖
- 3D 世界地圖

地圖網址由管理員提供。

目前 BlueMap 使用 Cloudflare Quick Tunnel，因此注意：

- 地圖網址可能會變更。
- 如果管理員重開 Cloudflare Tunnel，會產生新的地圖網址。
- 如果地圖網址無法開啟，請向管理員索取新的網址。

---

## 4. 出生點與保護區

伺服器已設定出生保護區：

```text
spawn
```

在出生區內：

- 玩家不能破壞方塊
- 玩家不能放置方塊
- 玩家不能 PvP 打架
- 怪物不會自然生成
- TNT 不會生效
- 苦力怕爆炸不會破壞地形
- 其他爆炸不會破壞地形

這個區域是公共安全區，請不要嘗試破壞或繞過保護。

---

## 5. /spawn 回出生點

玩家可以輸入：

```text
/spawn
```

回到伺服器出生點。

如果玩家有設定床或重生錨，死亡後會依伺服器設定優先使用床或重生錨。

---

## 6. GSit 坐下系統

玩家可以使用以下指令：

### 坐下

```text
/sit
```

### 躺下

```text
/lay
```

### 爬行

```text
/crawl
```

### 趴下

```text
/bellyflop
```

部分樓梯、半磚、地毯可直接互動坐下。

---

## 7. 浮空文字與 NPC

出生點設有伺服器資訊浮空文字與 NPC。

玩家可以查看：

- 伺服器規則
- Java / Bedrock 連線資訊
- 常用指令
- BlueMap 地圖說明

部分 NPC 可右鍵互動取得伺服器資訊。

---

## 8. 睡覺跳過夜晚

伺服器支援「少量玩家睡覺即可跳過夜晚」的設定。

若晚上太危險，請有床的玩家協助睡覺。

---

## 9. 領地與建築保護說明

目前伺服器沒有安裝玩家自助圈地插件。

也就是說，目前沒有以下功能：

- 玩家自行圈地
- 玩家自行設定領地權限
- 玩家自行信任朋友
- 玩家自行解除領地

目前的保護方式是：

- 出生點由 WorldGuard 保護
- 玩家破壞紀錄由 CoreProtect 記錄
- 發生偷竊或破壞時，請通知管理員查詢與回復

請尊重其他玩家的建築與物品。

---

## 10. 伺服器規則

禁止：

- 偷竊
- 惡意破壞建築
- 外掛
- 洗頻
- 惡意熊服
- 惡意卡服
- 在公共區域惡意干擾其他玩家

所有方塊破壞、方塊放置與部分互動紀錄皆由 CoreProtect 記錄。

若發生偷竊或破壞，管理員可以查詢並回復。

---

## 11. 問題回報

若發現以下問題，請通知管理員：

- 地圖無法開啟
- 建築被破壞
- 箱子物品遺失
- 玩家卡住
- 伺服器異常
- 出生點保護區失效
- 無法加入伺服器
- NPC 或浮空文字顯示異常
